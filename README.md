# Khalid-Fahd-BREP-Gradient-Porosity-Rhinoscript
Primarily contains rhinoscript code from a hip implant CAD project, enabling porosity gradients across a closed BREP surface, useful for biomaterial design, towards maximal osseointegration. The script takes user-input like filament/fiber radii, porosity bounds; min-max (in %) along the z-axis
